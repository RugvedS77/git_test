# Simple Interest Calculator

A lightweight, user-friendly tool designed to calculate the interest earned or paid on a principal amount over a specific period at a fixed interest rate.
## Features
Instant Calculation: Get results immediately after entering data.Error Handling: Validates input to ensure numerical values and positive timeframes.
Clean Interface: Minimalist design for ease of use.## How It WorksThe calculator uses the standard mathematical formula for simple interest
